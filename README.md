# tiny-chat
A minimal TCP-based chat system implemented in Go for learning purposes.

> [!NOTE]
> This is a toy implementation designed to understand TCP networking fundamentals and Go's `net` package.
> 
## Running the project
The server and client are separate programs that must be run in different terminals.

Start the server:
```bash
go run src/server.go
```

In another terminal, start the client:
```bash
go run src/client.go
```
